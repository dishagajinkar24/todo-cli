# To-Do CLI App üìù

A simple Command Line Interface (CLI) To-Do application built with Python.  
It allows users to add tasks, view tasks, and mark tasks as done. Tasks are saved locally in `tasks.json`.

---

## Features
- Add new tasks
- View all tasks
- Mark tasks as done
- Persist tasks to `tasks.json`

---

## How to run (Windows)
1. Clone the repo (if not already):
   ```
   git clone https://github.com/dishagajinkar24/todo-cli.git
   ```
2. Go into the folder:
   ```
   cd todo-cli
   ```
3. (Optional) Create virtual environment:
   ```
   python -m venv venv
   ```
4. Activate venv:
   ```
   venv\Scripts\activate
   ```
5. Run the app:
   ```
   python todo.py
   ```
